# Minesweeper in Java

This is my final project for a introductory java class.

## How to install ubuntu/debian
First,
```
sudo apt-get update
```
then
```
sudo apt-get install openjdk-8-jre
```
and
```
sudo apt-get install openjdk-8-jdk
```
now navigate to `java_minesweeper/src/`

and type

```
javac *.java
```
then run with
```
java Launcher
```
